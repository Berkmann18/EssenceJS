Run HTML5 canvas commands in a simpler and easier way.
Parameters:
Name Type Attributes Default Description
commands function Commands to execute on the canvas
dimension string <optional>
 "2d" Dimension
stackLayer number <optional>
 0 Stack layer
Since:
  • 1.1
Source:
  • modules/UI.js, line 1571
Returns:
undefined
Example
runCanvas(function (context) {
	   context.fillStyle = "#000";
	   context.strokeStyle = "#0f0";
	   context.fill();
	   context.stroke();
	   context.beginPath();
	   context.moveTo(0, 100);
	   context.lineTo(50, 0);
	   context.lineTo(100, 100);
	   context.lineTo(0, 100);
	   context.closePath()
});
//This will draw a black triangle with green borders
